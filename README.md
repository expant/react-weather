# Weather

A simple, user-friendly weather application that provides real-time forecasts for any city

### 🛠️ Technology Stack

- React
- Vite

### ⚙️ Installation

```bash
git clone https://github.com/expant/react-weather.git
cd react-weather
npm install
```

### 💻 Development

```bash
npm run dev
```

### 🏗️ Production Build

```bash
npm run build
npm run preview
```
